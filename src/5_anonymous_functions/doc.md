# 無名関数

関数定義

```elixir
sum = fn (a,b) -> a + b end
sum.(1,2)
> 3
```

無名関数を呼び出す時には `.`が必要。
名前付き関数の時には不要

関数が引数を取らなくても呼び出しにはカッコが必要

```elixir
greet = fn -> IO.puts("Hello") end
greet.()
```

関数定義の時にはパラメーターを囲むカッコは省略可能

```elixir
f1 = fn a,b -> a * b end
```

sum.(2,3) という呼び出しで `a,b というパラメーターに 2,3 が代入される｀　という理解は誤り。

**elixir には代入はない**

a,b に 2,3 をパラメーターマッチさせている。
